## Linux

### 命令行

#### 1. 查看Linux命令帮助信息
1. 查看Shell内部命令的帮助信息-使用help
for example
```bash
help #查看所有内建命令列表
help cd #查看某个内建命令的帮助
```
> 判断一个命令是内建命令还是外部命令的一个简单方法是使用type命令:
> ```bash
> type cd #如果是内建命令,会显示"cd is a shell builtin"
> type ls #如果是外部命令,会显示"ls is /bin/ls
> ```

2. 查看命令的简要说明-使用whatis
3. 查看命令的详细说明-使用info
for example
```bash
whatis cd #查看cd命令的简要说明
info cd #查看cd命令的详细说明
```
4. 查看命令的位置-使用which
5. 定位指令的二进制程序,源代码文件和man手册页等相关文件的路径-使用whereis
6. 查看命令的帮助手册(包含说明、用法等信息)-使用man

#### 2. Linux文件目录管理

##### 2.1 Linux文件目录工作机制

###### 2.1.1 Linux目录结构
Linux目录结构是树形结构,其根目录是/,各个目录的作用:

###### 2.1.2 Linux文件属性
Linux 系统是一种典型的多用户系统,不同的用户处于不同的地位,拥有不同的权限.为了保护系统的安全性,Linux 系统对不同的用户访问同一文件(包括目录文件)的权限做了不同的规定.在 Linux 中我们可以使用 ll 或者 ls –l 命令来显示一个文件的属性以及文件所属的用户和组.

##### 2.2 Linux文件目录管理要点

###### 2.2.1 目录管理
- 切换目录-使用cd
- 查看目录信息-使用ls
- 显示当前目录的绝对路径-使用pwd
- 树状显示目录的内容-使用tree
- 创建目录-使用mkdir    
- 删除目录-使用rmdir

###### 2.2.2 文件管理
- 创建空文件-使用touch
- 为文件创建连接-使用ln
- 批量重命名-使用rename
- 显示文件的详细信息-使用stat
- 探测文件类型-使用file
- 设置文件或目录的权限-使用chmod
- 设置文件或目录的拥有者和所属群组-使用chown
- 查找文件或目录-使用locate
- 在指定目录下查找文件或目录-使用find
- 查找命令的绝对路径-使用which
- 查找命令的程序,源代码等相关文件-使用whereis
